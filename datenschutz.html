Wo deine Daten liegen

Alle Trainingsdaten (Sätze, Gewichte, Körpergewicht, Schlaf, Energielevel) werden ausschließlich im lokalen Speicher deines Browsers (localStorage) auf diesem Gerät abgelegt. Wichtig: Löschst du den Browser-Cache/die Website-Daten für diese Seite oder setzt du dein Gerät zurück, werden diese Daten unwiderruflich gelöscht — TRAIN hat keine Kopie und kann sie nicht wiederherstellen.

Nutze dafür regelmäßig die JSON-Sicherung (Einstellungen → Backup) — die Export-Datei landet auf deinem Gerät bzw. deiner eigenen Cloud, ebenfalls nie auf einem Server von uns.

Offline-Funktion (Service Worker)

TRAIN speichert App-Dateien technisch im Browser-Cache, damit die App auch offline funktioniert. Das ist eine rein technische Verarbeitung auf Basis unseres berechtigten Interesses an einer zuverlässig nutzbaren App (Art. 6 Abs. 1 lit. f DSGVO) — dieselbe Rechtsgrundlage wie bei der Nutzungs-Zählung unten.

Anonyme Nutzungs-Zählung

Für anonyme Nutzungs-Zählung (wie viele Personen die App öffnen) wird GoatCounter eingesetzt — cookielos, ohne Werbe-Tracking, ohne Weitergabe an Dritte. Dabei wird deine IP-Adresse kurz technisch verarbeitet, aber nicht dauerhaft gespeichert. Rechtsgrundlage hierfür ist unser berechtigtes Interesse an der Weiterentwicklung der App (Art. 6 Abs. 1 lit. f DSGVO). Schriftarten werden selbst gehostet, es findet kein Aufruf bei Google statt.

Hosting

Diese Seite wird über GitHub Pages (GitHub Inc./Microsoft Corporation) ausgeliefert. Beim Aufruf der Seite verarbeitet GitHub dabei automatisch deine IP-Adresse in Server-Logs — unabhängig von der GoatCounter-Zählung oben. Mehr dazu im GitHub Privacy Statement.

Was TRAIN verarbeitet — und was nicht

  • ✔ Trainingsdaten, Körpergewicht, Schlaf, Energielevel (nur lokal auf deinem Gerät)
  • ✔ Anonyme, aggregierte Nutzungs-Zählung (GoatCounter)
  • ✘ Kein Standort, keine Kontakte, keine Kamera/Mikrofon
  • ✘ Keine Werbung, kein Werbe-Tracking, kein Verkauf von Daten
  • ✘ Keine Übertragung deiner Trainingsdaten an einen Server
